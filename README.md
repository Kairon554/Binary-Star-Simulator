\# Binary Star System Simulator — Six Orbital Scenarios



A numerical simulation of the two-body problem using Python, NumPy, and Matplotlib.

This project visually demonstrates \*\*Kepler's Three Laws\*\*, \*\*orbital precession\*\*, and \*\*gravitational slingshot (hyperbolic escape)\*\*.



\---



\##  Six Scenarios



| Scenario | Description | Key Physics Concept |

|----------|-------------|----------------------|

| 1 | Equal-mass binary stars (elliptical orbits) | Center of mass, symmetric orbits |

| 2 | Sun-planet system (mass ratio 5:1) | Kepler's laws, stellar wobble |

| 3 | Extreme mass ratio (1:10000) | Orbital precession (like Mercury's perihelion shift) |

| 4 | High-eccentricity ellipse (Halley's comet style) | Fast near perihelion, slow near aphelion |

| 5 | Hyperbolic escape (gravitational slingshot) | Escape velocity, open orbits |

| 6 | Perfect circular orbit | Verification of circular velocity formula |
